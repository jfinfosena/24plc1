## PLC

### ¿Qué es un PLC?

Un PLC (Controlador Lógico Programable) es un tipo de computadora especializada diseñada para controlar procesos industriales y automáticos.  Estos dispositivos reciben información de sensores (entradas) y la procesan para generar acciones en actuadores (salidas), como encender motores, abrir válvulas o activar luces. 

### Historia y evolución de los PLC

* Década de 1960: Nacen los primeros PLC como una alternativa a los sistemas de control basados en relés, que eran complejos y difíciles de modificar. 
* Década de 1970: Se popularizan los PLC, con mejoras en la capacidad de procesamiento y la facilidad de programación.
* Década de 1980:  Aparecen los PLC con interfaces de usuario más amigables, lenguajes de programación más potentes y mayor conectividad.
* Década de 1990: Se introducen los PLC con capacidad de comunicación Ethernet, lo que permite la integración con sistemas de control de nivel superior.
* Actualidad: Los PLC modernos son sistemas robustos, con capacidades de procesamiento avanzadas, interfaces gráficas de usuario intuitivas y una amplia gama de opciones de comunicación.

### Ventajas de los PLC

* Flexibilidad: Fáciles de programar y reprogramar para adaptarse a diferentes aplicaciones.
* Fiabilidad: Diseñados para funcionar en entornos industriales difíciles y resistir a la vibración, el polvo y la humedad.
* Robustez: Resistentes a interferencias electromagnéticas y a las variaciones de temperatura.
* Facilidad de mantenimiento: Diagnóstico sencillo y piezas de repuesto fáciles de encontrar.
* Costo-efectivos:  A largo plazo, los PLC suelen ser más económicos que los sistemas de control tradicionales.

### Aplicaciones de los PLC en la industria

Los PLC se utilizan en una gran variedad de industrias, incluyendo:

* Automatización de procesos: Control de líneas de producción, llenado de botellas, empaquetado, etc.
* Control de máquinas:  Manejo de robots, máquinas de corte, máquinas herramienta, etc.
* Control de edificios:  Sistema de calefacción, ventilación y aire acondicionado, iluminación, seguridad, etc.
* Automatización de procesos químicos:  Control de reacciones químicas, procesos de separación, etc.
* Automatización de sistemas de energía:  Control de turbinas, generadores, sistemas de distribución eléctrica, etc.

## Arquitectura de un PLC

### Componentes principales de un PLC

* CPU (Unidad de Procesamiento Central): El "cerebro" del PLC, que procesa la información, ejecuta el programa y controla las entradas y salidas.
* Módulos de entrada:  Reciben señales de sensores, como interruptores, detectores, sensores de temperatura, etc.
* Módulos de salida: Controlan actuadores, como motores, solenoides, válvulas, lámparas, etc.
* Memoria:  Almacena el programa, los datos y la configuración del PLC.
* Fuente de alimentación:  Suministra energía al PLC.
* Interfaz de usuario: Permite al usuario programar, configurar y monitorear el PLC.

### Unidad de Procesamiento Central (CPU)

* Procesamiento de información: La CPU interpreta las señales de entrada, ejecuta el programa y genera las señales de salida.
* Control del programa:  La CPU controla el flujo del programa, ejecuta las instrucciones y actualiza la memoria.
* Comunicación:  La CPU se comunica con otros dispositivos, como computadoras, otros PLC, etc.

### Módulos de entrada y salida

* Entradass: Reciben señales de sensores y las convierten a un formato digital que la CPU puede entender.
* Salidas:  Convierten señales digitales de la CPU a señales analógicas para controlar actuadores.

### Memoria del PLC

* Memoria de programa: Almacena el programa del PLC.
* Memoria de datos:  Almacena los valores de las variables, los datos del proceso y la configuración del PLC.
* Memoria de retención: Almacena los datos críticos que se deben conservar incluso cuando el PLC se apaga.

### Interfaz de usuario y programación

* Panel de operador:  Una pantalla que muestra el estado del PLC y los valores de las variables.
* Software de programación:  Permite al usuario crear, editar y cargar programas al PLC.
